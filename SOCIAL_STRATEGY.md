# Strategie social media Bazario

## Plateformes prioritaires

1. TikTok : decouverte produit, formats courts, UGC, tests de tendances.
2. Instagram : reels, stories, preuve sociale, messages directs support.
3. Pinterest : trafic long terme pour maison, mode, cadeaux et inspirations.

## Ligne editoriale

- Ton : direct, utile, francophone, transparent sur les delais et la selection.
- Promesse : "Tout, mieux, partout" avec preuve concrete.
- Formats : demonstrations, avant/apres, problemes du quotidien, coulisses, avis clients.

## Influence

- Cible : nano et micro-influenceurs francophones (1k a 50k abonnes).
- Criteres : audience FR, commentaires authentiques, niche claire, historique de placements raisonnable.
- Offre : gifting sur produits reels, code promo nominatif, commission a definir apres Stripe/checkout reel.

### Message de contact

Bonjour [prenom],

Je lance Bazario, une selection e-commerce francophone orientee produits utiles et transparence client. J'aime votre contenu sur [theme] et je pense qu'une collaboration simple pourrait interesser votre audience.

Seriez-vous ouvert(e) a tester un produit et partager un retour honnete ? Nous pouvons fournir un code promo dedie et suivre les performances.

Merci,
L'equipe Bazario

## Metriques

- Portee, taux de retention video, clics profil, clics site.
- Taux d'inscription newsletter.
- Ajouts panier, commandes, chiffre d'affaires par code promo.
- Cout par contenu teste et taux de reutilisation UGC.
