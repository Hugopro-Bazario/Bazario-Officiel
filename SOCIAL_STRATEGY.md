# Stratégie réseaux sociaux + influence — Bazario

## 1. Plateformes prioritaires

## 2. Charte éditoriale

- **Ton** : chaleureux, expert, direct. Pas de jargon. On tutoie.
- **Sujets** : produits hero, coulisses vendeurs, conseils pratiques, UGC clients, drops/soldes.
- **Identité visuelle** : photo lumière naturelle, palette ivoire/charbon/laiton, typo Geist, jamais de stock photo générique.
- **Fréquence** :
  - Instagram : 1 post + 3 stories + 2 Reels / jour
  - TikTok : 1-2 vidéos / jour
  - YouTube : 1 long form / semaine + 3 Shorts
  - Pinterest : 5-10 pins / jour (tableaux thématiques)
  - LinkedIn : 2 posts / semaine
  - X : 5-7 tweets / jour

## 5. Stratégie d'influence

### 5.1 Type d'influenceurs cibles

- **Phase 1 (Mois 1-3)** : nano (1-10 k abonnés) et micro (10-100 k). Engagement supérieur, coût modéré, authenticité.
- **Phase 2 (Mois 4-9)** : mid-tier (100 k - 1 M) sur niches précises (déco, mode lente, lifestyle, parents).
- **Phase 3 (Mois 10+)** : macro (1 M+) pour notoriété, 1-2 partenariats hero par trimestre.

### 5.2 Critères de sélection

- Taux d'engagement > 3 % (nano) / > 2 % (micro) / > 1.5 % (mid)
- Audience à 70 %+ francophone
- Cohérence éditoriale avec Bazario (pas de fast-fashion ultra)
- Pas plus de 2 partenariats sponsorisés sur les 7 derniers jours
- Vérification compte non gonflé via HypeAuditor / Modash

### 5.3 Modèles de contact (DM Instagram)

```
Salut {prenom} !
On adore ton compte, particulièrement ton post sur {sujet}.
Bazario est une marketplace francophone qui sélectionne uniquement des vendeurs
vérifiés (aujourd'hui 12 800 dans 42 pays). On voudrait t'envoyer 2-3 pièces
de notre catalogue {categorie}, sans engagement de poster, juste pour que
tu te fasses ton avis. Ça te dirait ?

Ton équipe
Bazario
```

### 5.4 Conditions de partenariat

- **Gifting** : nano + micro qualifiés, pas d'engagement ferme. Code de réduction tracké.
- **Code promo + commission affiliation** : 10 % commission sur ventes, code -10 % pour leur communauté.
- **Brand deal** : mid-tier et macro, devis sur mesure (200 € - 5 000 €), 3-5 livrables (1 Reel, 3 stories, 1 carrousel).
- **Ambassadeurs** : top 5 performers par trimestre, contrat annuel, abonnement Bazario+ offert + 15 % commission à vie.

## 7. Outils recommandés

- **Programmation** : Buffer (3 plateformes gratuit) ou Later (Instagram-first)
- **Analytics** : Iconosquare ou Sprout Social
- **Influence** : Modash, HypeAuditor, Heepsy
- **UGC** : Insense, Billo (vidéos UGC à la demande)
- **Édition vidéo** : CapCut Pro, Descript
- **Design** : Figma + templates Canva
- **Liens en bio** : Linktree ou page native Bazario `/links`
